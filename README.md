Chatbot Web – Patoche:

Chatbot web full-stack développé dans un cadre académique, combinant un backend Django et un frontend React, avec intégration d’un modèle de langage (LLM) via API.
Le projet est déployé en production et accessible en ligne.

Fonctionnalités:

API backend exposant un endpoint /ask pour le traitement des messages

Intégration d’un LLM via API (Poe / Claude)

Gestion des erreurs avec réponses de secours (fallback) en cas d’indisponibilité du modèle

Sécurisation des accès (CORS, CSRF trusted origins, ALLOWED_HOSTS)

Interface utilisateur type messagerie instantanée (style WhatsApp)

Application déployée et accessible en ligne
